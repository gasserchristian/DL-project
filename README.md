## Git

**Commands for pushing**

```shell
git clone git@github.com:gasserchristian/DL-project.git
cd DL-project
git commit -m "message"
git pull
git push

# list remote branches
git branch -r
# list local branches, 
git branch

# first fetch if want to checkout to a remote branch
git fetch
git checkout <branch name>
```
